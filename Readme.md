# AI Finance Snapshot Bot

A Telegram bot for:
- Analyzing personal finance data from Excel files
- Extracting transactions from receipt images via OCR
- Parsing text-based transactions
- Supporting bilingual UI (FA/EN)

## Setup

```bash
git clone <repo-url>
cd AI_Finance_Bot
python -m venv venv
venv\Scripts\activate  # On Windows
pip install -r requirements.txt
